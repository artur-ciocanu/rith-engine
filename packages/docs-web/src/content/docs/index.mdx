## Install in seconds

:::code-group

```bash [macOS / Linux]
curl -fsSL https://github.com/artur-ciocanu/rith-engine/install | bash
```

```powershell [Windows]
irm https://github.com/artur-ciocanu/rith-engine/install.ps1 | iex
```

```bash [Homebrew]
brew install coleam00/rith/rith
```

```bash [Docker]
docker run --rm -v "$PWD:/workspace" ghcr.io/artur-ciocanu/rith-engine:latest workflow list
```

:::

## What is Rith Engine?

Rith Engine is a **workflow engine for AI coding agents**. Define multi-step development workflows in YAML — code review, bug fixes, feature implementation, testing — and run them with a single command.
